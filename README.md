# geogebra - linux - standalone

just the original geogebra packaged as linux appimage


# technologies used
* electron
* vite
* node.js
* vue.js
* bootstrap.css

## install necessary modules 

```npm i```

## run geogebra

```npm run dev```

## build geogebra

```npm run build```
```npm run build:win```
```npm run build:mac```
