# geogebra - linux - standalone

just the original geogebra packaged as linux appimage and debian package


# technologies used
* electron
* vite
* node.js
* vue.js
* bootstrap.css

## install necessary modules 

```npm i```

## run geogebra

```npm run dev```

## build geogebra for linux (appimage, deb)

```npm run build```
